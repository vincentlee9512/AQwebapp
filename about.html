Project Tools

We have used frameworks and other tools to make the building of this app more flexible.

Bootstrap

Bootstrap was used to expedite the creation of our app. This allows for use of templates that make development quicker.

Bootswatch Theme CDN

We are leveraging the power of a CDN in order to load our base theme. A CDN offers fast load time for a user, especially when it is cached. Since many websites are using Bootstrap and Bootswatch themes, performance gains are optimal.

Google Maps API

Google Maps API was essential to producing the Web App. This allowed for the map interface, as well as certain features within the map, such as clusters.

Open AQ API

The Open AQ API was used to overlay air quality measurements on the map. Air quality data for certain coordinates (based on user input) were gathered and displayed in a table below the map.

Angular JS

We used angular to speed up development, as well as make it easier to handle the Open AQ data. The most prominent example of this is using: ng-repeat. This allows us to go through the data and parse it to rapidly populate the air quality table.
